# Генератор QR-кодов

Простой статический сайт на базе `qrcode.js`.

**Возможности:**
- Генерация QR-кода по ссылке.
- Автоматическое добавление `https://`.
- Очистка UTM-меток.
- Скачивание в `PNG`, `SVG`, `JPG`.

## Запуск

1. Локально: откройте `index.html`.
2. Online: используйте GitHub Pages (Source: `Deploy from a branch`, Branch: `main`).
